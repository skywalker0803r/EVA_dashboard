## 地址
https://lab.magiconch.com/magi/

## 如何使用
点击鼠标 或者 敲击空格 即可开启决议

## 功能
- [x] 基础决议
- [x] EX MODE
- [x] 概率调整
- [x] 优先级
- [x] 提示音效
